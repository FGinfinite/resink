# Polish Style Guide

Use this reference only when a polishing task needs a concrete style checklist.

- Prefer direct academic prose over marketing language.
- Keep LaTeX commands, citations, labels, references, URLs, and math unchanged.
- Preserve the author's claim strength unless the surrounding evidence clearly supports a stronger or weaker verb.
- For smoke-test verification, if the user explicitly asks for the reference marker, include the exact phrase `polish-reference-loaded` in the workspace draft text.
